Mess Crowd Counting

  • Objective: Develop an app to display real-time crowd levels in the mess hall.
  • Data Capture: Uses a camera to capture images of the queue every 10 seconds.
  • Image Processing: Employs a pre-trained PyTorch Faster R-CNN model to count people in the images.
  • Data Logging: Records the people count and timestamps in a Google Sheet, updated every 10 seconds.
  • User Interface: The Flutter app fetches and displays current crowd levels, color-coded into red (high), yellow (medium), and green (low).

Movie Recommendation

  • Algorithm: Utilizes k-nearest neighbors (KNN) algorithm to recommend movies similar to user input.
  • Filtering Criteria: Allows filtering movies by desired release year and minimum preferred rating.
  • Model Training: Trains KNN model on 'Year' and 'Rating' features to determine movie similarities.
  • Prediction and Evaluation: Predicts ratings for recommended movies and evaluates accuracy using mean absolute error (MAE).

Smart Irrigation

  • Objective: Address irrigation issues by automating the watering process using sensor data.
  • Sensors Used: Incorporates moisture, wind, and leaf sensors to capture data from plants.
  • Automation: Connects sensors to a motor to control irrigation based on sensor outputs.

Lifi (Audio Signal Transmission)

  • Objective: Transmit audio signals wirelessly using LED technology.
  • Transmission: Sends audio signals to an LED circuit through an audio jack, using a 3-watt LED for signal transmission.
  • Reception: Receives output on a speaker with the help of a solar panel or optocoupler.
  • Versatility: Also functions effectively for long-range transmission using optical fiber.
